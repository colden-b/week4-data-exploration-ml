## Assignment Overview

This project applies data exploration, feature engineering, and basic modeling using Python and scikit-learn, using the **Wisconsin Breast Cancer dataset** to predict whether a tumor is malignant or benign.

## How to Run

1. Open in Google Colab or Jupyter Notebook
2. Install dependencies: `pip install numpy pandas matplotlib scikit-learn`
3. Run all cells top to bottom

## Dataset

Breast Cancer Wisconsin — built into scikit-learn. 569 samples, 30 features, binary target (malignant/benign).

## Results

| Model | Accuracy |
|---|---|
| Logistic Regression | ~96% |
| Random Forest | ~97% |
